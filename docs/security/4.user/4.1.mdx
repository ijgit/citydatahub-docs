# 4.1. 일반 사용자 계정 생성

데이터 허브의 내부 모듈의 사용, 외부 어플리케이션을 등록하기 위해서는 데이터 허브에 회원가입을 진행해야 합니다.

1. 인증/인가 모듈의 GUI 메인 페이지에 접속하여 하단의 **"회원가입"** 버튼을 클릭합니다.

    <div style={{textAlign: 'center'}}>
    <img src={require('./4.1_image/1.Main_Page.png').default} width="400"/>
    </div>

2. **이용약관 및 개인정보 수집 및 이용안내**를 읽으시고 체크박스를 클릭하여 동의를 합니다.

    <div style={{textAlign: 'center'}}>
    <img src={require('./4.1_image/2.Enrollment.png').default} width="400"/>
    </div>


3. 사용자의 필수 정보들을 입력하고 **"회원가입"** 버튼을 클릭합니다.

    <div style={{textAlign: 'center'}}>
    <img src={require('./4.1_image/3.Enrollment_two.png').default} width="400"/>
    </div>
   

4. 회원가입이 완료되었습니다. 이제 데이터 허브의 내부 모듈 이용 및 외부 어플리케이션 등록이 가능합니다.

    <div style={{textAlign: 'center'}}>
    <img src={require('./4.1_image/4.Enrollment_complete.png').default} width="400"/>
    </div>
