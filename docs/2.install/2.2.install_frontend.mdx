# 2.2 Frontend 설치

인증/인가 모듈의 Frontend는 인증/인가 모듈의 API를 이용하기 위한 GUI입니다.

React로 구현되있으며 2.1 Backend 설치에서 node 및 Npm을 설치하셨으면 무리 없이 설치가 가능합니다.



## 2.2.1 Node_modules 설치

설치를 진행하기 전 관리자 계정으로 변경합니다.
```bash
[centos@localhost smartcity_back]$ sudo su - root
[root@localhost ~]#
```

<br/>

프론트엔드의 디렉토리로 이동합니다.
```bash
[root@localhost smartcity_back]# cd ../smartcity_front/
```

<br/>

npm을 이용하여 필요한 노드모듈을 설치합니다. 이 때 리액트도 자동으로 설치됩니다.
```bash
[root@localhost smartcity_front]# npm i
```

<br/>

프론트엔드를 원하는 포트에서 실행하기 위해 `package.json`의 수정이 필요합니다. `package.json`의 "scripts" 의 "start" 부분을 수정합니다.

```bash
[root@localhost smartcity_front]# vim package.json
```

<br/>

30001번 포트에서 구동할 경우, 아래와 같이 변경합니다.

```json title="package.json" {2}
"scripts":{
    "start": "export PORT=30001 && react-scripts start",
    "build": "react-scripts build",
    "test": "react-scripts test",
    "eject": "react-scripts eject"
}
```

<br/>

설정이 완료되었으면 2.1 Backend 설치 과정에서 설치된 pm2를 이용하여 실행합니다.

```bash
# root 계정이 아니라면 root 계정으로 전환합니다.
# [centos@localhost smartcity_back]$ sudo su - root

[root@localhost smartcity_front]# pm2 start npm -- start
```

## 2.2.2 관리자 비밀번호 변경

2.1 Backend 설치 과정을 하였으면 디폴트 관리자 계정인 admin 계정이 생성된 상태입니다.
프론트엔드를 이용하여 admin 계정으로 로그인합니다.
로그인 후 **내 정보 보기 버튼**을 클릭합니다.

:::note
admin 계정의 초기 비밀번호는 admin45@ 이므로 원하는 비밀번호로 변경하여 사용하시기 바랍니다.
:::
