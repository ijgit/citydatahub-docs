# Install Smartcity Backend 

:::caution
인증/인가 모듈의 Backend 입니다. 실질적인 인증/인가를 담당하며 <br/>
Node 버전 12, npm v6 이상을 권장합니다.
:::

## Node js 및 Postgresql 설치

 2. `sudo yum update`
